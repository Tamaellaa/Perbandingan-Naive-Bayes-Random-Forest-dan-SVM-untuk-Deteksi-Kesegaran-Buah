# Perbandingan-Naive-Bayes-Random-Forest-dan-SVM-untuk-Deteksi-Kesegaran-Buah
Perbandingan Naive Bayes, Random Forest, dan SVM untuk Deteksi Kesegaran Buah Menggunakan Ekstraksi Fitur Warna dan Tekstur
Proyek ini membandingkan tiga algoritma machine learning — **SVM, Random Forest, dan Naive Bayes** — untuk mengklasifikasikan kesegaran buah (fresh vs rotten) menggunakan ekstraksi fitur warna dan tekstur dari citra.
Link Dataset : https://www.kaggle.com/datasets/narayanibokde/augmented-dataset-for-fruits-rottenfresh
